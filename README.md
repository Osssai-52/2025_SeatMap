# SeatMap - 강의실 좌석 점유 실시간 탐지 앱

> 2025 공과대학 10주년 기념 학술대회 | 팀 SM모델공방

강의실의 좌석 점유 상태를 **YOLOv8 기반 딥러닝 모델**로 실시간 감지하고, **Flutter 앱**에서 직관적으로 시각화하는 서비스입니다.

---

## 프로젝트 배경

| 문제 | 설명 |
|------|------|
| 빈 강의실 탐색 어려움 | 강의실 예약은 3일 전 신청이 규정이라, 당장 필요한 경우 직접 돌아다니며 확인해야 함 |
| 대형 강의 빈 좌석 탐색 | 수강생이 많은 강의에서 빈 자리를 한눈에 찾기 어려움 |
| 에너지 낭비 | 빈 강의실에 조명·냉방이 켜져 있어도 확인이 번거로움 |

> 재학생 대상 설문조사 결과, **93.3%**가 빈 강의실 찾기에 어려움을 경험했고, **82.2%**가 본 서비스 사용 의향이 있다고 응답했습니다.

---

## 주요 기능

- **실시간 감지** — YOLOv8 모델로 좌석 점유 여부 판별
- **좌석 시각화** — 앱에서 좌석 배치도를 색상(🔴 점유 / ⚪ 빈 좌석)으로 표현
- **어디서나 확인** — 모바일 앱으로 언제든 강의실 좌석 현황 조회
- **에너지 절약 알림** — 모든 좌석이 비어 있으면 '에어컨을 꺼주세요' 메시지 표시

---

## 시스템 아키텍처

```
📱 휴대폰 촬영 (Larix Broadcaster)
    ↓ RTSP
🖥️ MediaMTX 스트리밍 서버
    ↓
🔍 OpenCV 전처리 + ROI 설정 → YOLOv8 추론
    ↓ JSON (좌석별 occupied/empty + confidence)
⚡ FastAPI + WebSocket
    ↓ 실시간 통신
📱 Flutter 앱 시각화
```

## 📊 모델 성능

| Class | Images | Instances | Precision | Recall | mAP50 | mAP50-95 |
|-------|--------|-----------|-----------|--------|-------|----------|
| **all** | 25 | 702 | 0.874 | 0.865 | **0.904** | 0.564 |
| empty | 25 | 440 | 0.852 | 0.848 | 0.889 | 0.564 |
| occupied | 25 | 262 | 0.895 | 0.883 | 0.919 | 0.564 |

- Precision / Recall / mAP50 모두 100 에폭 내에서 빠르게 수렴
- Training Loss와 Validation Loss가 안정적으로 감소하여 과적합 없이 학습 완료

## 앱 화면

- **Home** — 명신관 5층 강의실 배치를 반영한 강의실 선택 화면
- **좌석 현황** — 점유 좌석(빨강) / 빈 좌석(회색) 색상 구분
- **에너지 절약** — 전체 빈 강의실 감지 시 '에어컨을 꺼주세요' 안내

## 향후 계획

- **에너지 절감 연동** — 20분 이상 무점유 시 시설 관리팀에 자동 알림
- **다강의실 확장** — 독립 파이프라인 + 중앙 서버 통합 아키텍처
- **학사시스템 연동** — 시간표·예약 정보와 실제 점유 상태 비교
- **다중 센서/카메라** — 사각지대 해소를 위한 추가 카메라 및 IoT 센서 도입
- **UI 개선** — 관리자/사용자 화면 분리, 좌석 상태 세분화(사람/짐/빈 좌석)
